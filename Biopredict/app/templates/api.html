Bio-Prodict Chromosome Variant API

This is page only shows information about the single argument url based api.


Usage of the API:
Format:
chr=&loc=&var=
chr: Chromosome, loc: Location, var: Variance

Example usage:
/api?chr=7&loc=195751&var=A

Usage of the url based api returns json.


Errors are represented as raw text
